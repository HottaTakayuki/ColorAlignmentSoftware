### 1-2. モジュール命名規約

| 項目 | 規約 |
|------|------|
| 命名方針 | クラス/メソッドは PascalCase、イベントは control_event 形式 |
| ID採番規則 | MDL-GAP-001 から連番 |
| 分類コード | SCR:画面, BIZ:ビジネスロジック, DAL:データアクセス, IF:外部IF |

### 1-3. Verona/Capri向け追加責務

| 責務種別 | 実装モジュール | 詳細 |
|---------|--------------|------|
| gamma値取得 | MDL-GAP-003 | Controller別gamma値を ADCP JSON `CmdGammaValueAllGet` で一括取得、補正計算へ反映 |
| moire判定 | MDL-GAP-003 | 撮影画像を DFT解析で周波数成分チェック、`calcMoireCheckArea` / `checkMoire` で判定 |
| 距離連動露出補正 | MDL-GAP-002/003 | `SetCamPosTarget(..., zDistanceSpec)` で zDistanceSpec別に F値/SS を動的調整 |
| 反復補正評価 | MDL-GAP-004 | moire検出または規格外時に補正ループ継続、`Spec_by_Zdistance` で判定閾値切替 |
| ADCP一括I/F | MDL-GAP-005/006 | ADCP JSON形式で gamma取得、補正値一括設定、一括取得実装 |
| 自動書込み | MDL-GAP-006 | `Auto_WriteData` 指定時に補正完了直後に ROM書込み自動実行 |
