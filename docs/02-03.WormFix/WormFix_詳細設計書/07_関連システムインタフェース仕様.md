### 7-3. インタフェース処理シーケンス

#### 7-3-1. 補正値書き込み処理シーケンス

```mermaid
sequenceDiagram
    participant UI as WormFix UI
    participant ENG as WormFixEngine
    participant CTRL as Controller

    UI->>ENG: 補正/書き込み開始
    ENG->>CTRL: Panel OFF
    loop modified Cabinet
        ENG->>CTRL: Worm Write
    end
    ENG->>CTRL: Reconfig
    ENG->>CTRL: Panel ON
    CTRL-->>ENG: 応答
    ENG-->>UI: 完了/失敗通知
```

#### 7-3-2. 補正処理シーケンス

```mermaid
sequenceDiagram
    participant UI as WormFix UI
    participant ENG as WormFixEngine
    participant CTRL as Controller

    UI->>ENG: 補正処理開始
    ENG->>CTRL: 撮影条件設定
    loop all pattern
        ENG->>CTRL: 内蔵パターン表示
        CTRL-->>ENG: 応答
        ENG->>ENG: 撮影実行
    end
    ENG->>ENG: 画像取得
    ENG->>ENG: 画像解析/データ抽出
    ENG->>CTRL: 補正値書き込み
    CTRL-->>ENG: 応答
    ENG-->>UI: 補正結果通知
```
```mermaid
sequenceDiagram
    participant UI as WormFix UI
    participant ADJ as AdjustmentEngine
    participant MEAS as MeasurementEngine
    participant CTRL as Controller

    UI->>ADJ: 補正開始
    ADJ->>MEAS: 測定開始
    MEAS->>CTRL: 撮影条件設定
    loop all pattern
        MEAS->>CTRL: 内蔵パターン表示
        CTRL-->>MEAS: 応答
        MEAS->>MEAS: 撮影実行
    end
    MEAS->>MEAS: 画像取得
    MEAS->>MEAS: 画像解析/データ抽出
    ADJ-->>UI: 補正結果通知
```
