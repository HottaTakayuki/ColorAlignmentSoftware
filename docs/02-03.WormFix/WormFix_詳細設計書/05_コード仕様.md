
## 5. コード仕様

### 5-1. コード一覧

| コード名称 | コード値 | 内容説明 | 利用箇所 | 備考 |
|------------|----------|----------|----------|------|
| WormStatus | Before | 補正前表示状態 | 結果表示 | enum |
| WormStatus | Result | 補正後表示状態 | 結果表示 | enum |
| WormStatus | Processing | 補正処理中表示状態 | 結果表示 | enum |
| CaptureNum | 3 | 撮影回数 | 補正処理 | 定数 |
| WormTrimmingAreaMin | 50 | トリミング最小 | 解析処理 | 定数 |
| WormTrimmingAreaMax | 5000 | トリミング最大 | 解析処理 | 定数 |
| 補正初期値 | 128 | レジスタ初期値(100%相当) | 補正 | WormCellCorrectValue |
// WormFixはGapCameraのような一括補正・検出処理は持たず、UI操作・データ管理が中心

### 5-2. コード定義ルール

| 項目 | ルール |
|------|--------|
| 補正値範囲 | `correctValue_Min`〜`correctValue_Max` にクリップ |
| Cabinet識別変換 | `PortNo`,`UnitNo` をSDCPコマンドバイトへ変換 |
| 条件コンパイル | `CameraPosition`,`BulkSetCorrectValue`,`Auto_WriteData` など運用定義に従う |
