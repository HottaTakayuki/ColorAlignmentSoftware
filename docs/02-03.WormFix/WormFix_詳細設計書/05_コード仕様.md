## 5. コード仕様（btnWormCamAdjStart_Click専用）

### 5-1. コード一覧

| コード名称 | コード値 | 内容説明 | 利用箇所 | 備考 |
|------------|----------|----------|----------|------|
| WormFixStatus | Before | 補正前表示状態 | 結果表示 | enum |
| WormFixStatus | Measure | 計測表示状態 | 結果表示 | enum |
| WormFixStatus | Result | 補正後表示状態 | 結果表示 | enum |
| CaptureNum | 3 | 撮影回数 | 補正処理 | 定数（実装依存） |
| WormAreaThreshold | double | Worm判定面積比しきい値 | detectWormAsync | 設定値 |
| PortNo | int | SDCPコマンド用ポート番号 | UnitInfo等 | 1始まり |
| UnitNo | int | SDCPコマンド用ユニット番号 | UnitInfo等 | 1始まり |
| CellDataFile | string | セルデータファイル名 | UnitInfo | "uXXYY_cd.bin"形式 |
| UserSetting | struct | ユーザー設定情報 | Measurement.cs | ControllerID, LowBrightnessMode等 |

### 5-2. コード定義ルール

| 項目 | ルール |
|------|--------|
| Cabinet識別変換 | `PortNo`,`UnitNo` をSDCPコマンドバイトへ変換（UnitInfo, Cmd生成部参照） |
| ファイル命名規則 | セルデータ：`uXXYY_cd.bin`、エージングデータ：`uXXYY_ad.bin` |
| 補正値範囲 | Worm補正値はデバイス許容範囲へクリップして反映する |
| しきい値範囲 | R/G/Bしきい値は整数かつ設定範囲内のみ受理する |
| 測定データ保存 | 測定結果はXML/Excel形式で保存（saveMeasuredData, makeResultFileUnit等） |
| 条件コンパイル | `CameraPosition`,`BulkSetCorrectValue`,`Auto_WriteData` など運用定義に従う |
| ユーザー設定復帰 | 測定前後でUserSettingを保存・復帰（getUserSettingMeasure, setUserSettingMeasure） |
| 進捗・例外 | WindowProgressで進捗表示、例外はShowMessageWindowで通知 |

### 5-3. コード運用ルール

| 項目 | ルール |
|------|--------|
| CSV互換性 | WormFix専用CSVフォーマット以外は読込失敗として扱う |
| ログ粒度 | 開始/終了/主要分岐（CSV経路、測定経路、書込み可否）を最低限記録する |
| 書込み制御 | ユーザー確認で Yes の場合のみ `hc.bin` 書込み処理へ進む |
| 復帰処理 | 例外時は進捗UI、ボタン状態、ユーザー設定を必ず復帰する |
