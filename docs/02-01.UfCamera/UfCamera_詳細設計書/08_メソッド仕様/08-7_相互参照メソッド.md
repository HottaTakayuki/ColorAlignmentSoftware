### 8-7. 相互参照メソッド

本節の対象メソッドは、UfCamera 側では呼出し・連携責務のみを持ち、実装詳細の正本は
GapCamera 詳細設計書とする。

#### 8-7-1. 参照方針

| 項目 | 内容 |
|------|------|
| 目的 | GapCamera/UfCamera 間で重複記載を避け、Gap 側表示信号・SDCP 書込み系メソッドの詳細仕様を GapCamera 側へ集約する。 |
| UfCamera記載範囲 | 呼出し契機、利用箇所、前提条件への影響までを記載する。 |
| 詳細記載先 | GapCamera 詳細設計書（第8章） |

#### 8-7-2. 参照対象メソッド一覧

| No. | メソッド名 | UfCameraでの主な利用場面 | 参照先（GapCamera_詳細設計書） |
|-----|-----------|--------------------------|---------------------------------|
| 1 | outputIntSigWindowByController | Controller 単位の Window 信号表示 | 第8章 8-5-3 相当の表示連携処理 |
| 2 | outputIntSigFlatGap | Gap 補正確認用の FlatGap 表示 | 第8章 8-5-3 相当の表示連携処理 |
| 3 | setGapCvCell 系 | Gap 補正値の SDCP 書込み比較・整合確認 | 第8章 8-4 相当の補正/書込み処理 |

#### 8-7-3. メソッド別記載（参照定義）

##### 8-7-3-1. outputIntSigWindowByController

| 項目 | 内容 |
|------|------|
| UfCameraでの扱い | Controller 単位表示の利用契機と呼出し箇所のみ本書管理。 |
| 詳細仕様 | GapCamera 詳細設計書を参照。 |

##### 8-7-3-2. outputIntSigFlatGap

| 項目 | 内容 |
|------|------|
| UfCameraでの扱い | Gap 補正確認表示の利用契機のみ本書管理。 |
| 詳細仕様 | GapCamera 詳細設計書を参照。 |

##### 8-7-3-3. setGapCvCell 系

| 項目 | 内容 |
|------|------|
| UfCameraでの扱い | Gap 側 SDCP 書込み仕様との比較観点のみ本書管理。 |
| 詳細仕様 | GapCamera 詳細設計書を参照。 |
