## 6. メッセージ仕様 差分整理（ColorAlignmentSoftware_Nice）

## 要約

メッセージ仕様は主要な成功・失敗を押さえていますが、Nice 実装で増えた bulk 系メッセージと計測品質系エラーは未反映です。実装と合わせるには、正常系と異常系の一覧を拡張する必要があります。

## 差分一覧

| 観点 | 既存文書 | Nice 実装 | 影響 |
|------|----------|-----------|------|
| bulk restore | 未整理 | Restore(bulk) 成功/失敗メッセージがある | 情報・異常一覧へ追加が必要 |
| 計測品質エラー | 未整理 | moire 検出、camera position inappropriate、gamma 取得失敗がある | 異常系一覧の補強が必要 |
| backup/restore 整合 | 未整理 | backup value count mismatch のような整合エラーがある | 例外メッセージ補強が必要 |
| 通知経路 | 表示経路のみ簡略 | WindowMessage と ShowMessageWindow を使い分ける | 通知経路の明確化が必要 |

## 更新方針

- 情報、警告、異常の一覧に bulk 系と Nice 固有の例外文言を追加する。
- 画面通知経路に WindowMessage と ShowMessageWindow の使い分けを追記する。
