## 2. モジュール配置図 差分整理（ColorAlignmentSoftware_Nice）

## 要約

物理配置の骨格は維持できますが、Nice 実装では Controller 連携が SDCP だけでは閉じません。ADCP JSON 経路と計測条件依存の外部情報を追加しないと、実装構成図として不足します。

## 差分一覧

| 観点 | 既存文書 | Nice 実装 | 影響 |
|------|----------|-----------|------|
| Controller 経路 | SDCP のみ | ADCPClass 経由の JSON 通信を追加 | External と矢印の追加が必要 |
| 計測条件依存 | CameraControl / OpenCv / FileSystem 中心 | configurationOfLedModelList と Controller gamma 値が関与 | Measurement の依存先見直しが必要 |
| bulk 補正経路 | CameraControl 近傍の処理に見える | 実際は CameraControl とは別経路で Controller に入る | 配置理由の補強が必要 |

## 更新方針

- External に ADCPClass / JSON を追加する。
- Measurement / Adjustment から ADCP 経路を追加する。
- 配置理由に Nice の多 Controller 対応を追記する。
