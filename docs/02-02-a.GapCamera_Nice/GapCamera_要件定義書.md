# GapCamera 要件定義書 差分整理（ColorAlignmentSoftware_Nice）

## 要約

要件定義書で不足しているのは、Nice 実装で追加されている計測品質要件と Controller 一括連携要件です。既存文書は Gap 計測と補正の主経路は押さえていますが、moiré 判定、gamma 取得、ADCP 一括補正のような Nice 固有要素までは反映されていません。

## 差分一覧

| 観点 | 既存文書 | Nice 実装 | 影響 |
|------|----------|-----------|------|
| 計測要件 | Gap 計測・補正・ROM 書込み中心 | moire 判定も計測完了条件に含む | 2-4 機能要件、3章 品質要件の追記が必要 |
| 撮影条件 | 固定的な撮影条件前提 | 撮影距離入力に応じて F 値とシャッタ速度を切替 | 2-4、2-5 の前提値整理が必要 |
| 補正値 I/F | XML + SDCP 中心 | ADCP JSON による Controller 単位の一括取得・一括設定を追加 | 2-6 I/F 要件の拡張が必要 |
| 機種対応 | 抽象的なモデル差分 | VP15EB/EM, VP23EB/EM, VS25FM を含む個別分岐 | 2-2、2-8 の制約整理が必要 |

## 更新方針

- 2-4 機能要件に moire 検査、gamma 取得、撮影距離依存の露出補正を追加する。
- 2-5 データ要件に Controller gamma 値、撮影距離、moire 判定値を追加する。
- 2-6 関連 I/F 要件に ADCP JSON を追加する。
- 2-8 システム構築制約に Nice 固有の機種分岐と条件コンパイル影響を追記する。
